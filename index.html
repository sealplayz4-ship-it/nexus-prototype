Nexus - CS Benchmark

Benchmark your aptitude across core CS domains.
Compare your results to other students.

What this benchmark measures:
• Data structures and algorithms
• Systems reasoning
• Analytical Reasoning
• Code comprehension

Pseudocode conventions:
• Ranges use interval notation. For example from (1 to 3] does not include 1 but does include 3.
• Variables update sequentially
• Assume standard execution flow unless stated otherwise
• Variables are passed by reference unless otherwise stated.
• Assume loops terminate if condition becomes false after update

Please do not access outside resources when taking this test. Completion should take approximately 30 minutes. The assessment is experimental and feedback is appreciated.
